trejocode.css

Hoja de estilos basado en flexbox diseñado para realizar estructuras flexibles.

Básicos:

El uso de las propiedades de trejocode.css solo aplicarán si al contenedor padre le es asignado la clase .flex
Revisar trejocode.css antes de empezar las siguientes practicas, el código es completamente hackeable. 

                            
<div class="flex">

</div>

.container

El contenedor es un componente cuya función es cubrir por defecto el 90% del ancho. 

                            
<div class="flex">
    <div class="container">

    </div>
</div>

.auto

Auto establece que un elemento solo ocupe el ancho necesario. 

                            
<div class="flex">
    <div class="container">

    </div>
</div>
